## 🛠️ How to Add Animations (Best Practices for GitHub READMEs)

**1. Animated Typing Text (Recommended — Used above)**
- Visit [readme-typing-svg.demolab.com](https://readme-typing-svg.demolab.com)
- Customize text, colors, speed & copy the URL
- Paste the `<img>` tag anywhere in your README

**2. GitHub Stats Cards**
- Use [github-readme-stats](https://github.com/anuraghazra/github-readme-stats) (already included)
- Add theme like `tokyonight`, `radical`, `dark`

**3. Animated GIFs (Most Impactful)**
- Record screen demo of your projects (Terminal Lens or GuardianAI)
- Tools: 
  - **Loom** → export as GIF
  - **OBS Studio** + **ezgif.com**
  - **ffmpeg** (command-line)
- Embed like:
  ```md
  ![Demo](assets/terminal-demo.gif)
  ```

**4. CSS Animations (Advanced)**
Add this anywhere (GitHub supports limited HTML/CSS):

```html
<style>
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
.animated-card {
  animation: fadeIn 1s ease forwards;
}
</style>

<div class="animated-card">
  <!-- Content -->
</div>
```

**5. SVG Animations**
Use inline SVGs with `<animate>` tags for simple motion.
